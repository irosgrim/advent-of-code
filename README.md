
1. chmod +x ./solve_day.sh
2. chmod +x ./continue_day.sh

To start download the input run `./solve_day.sh 2022 1`

This will create a new folder in `src/2022/1` with the files needed to solve the challenge

To continue one existing day: `./continue_day.sh 2022 1`

Create `.env` file in the root of the folder:

```
AOC_SESSION_COOKIE=your session cookie
```
